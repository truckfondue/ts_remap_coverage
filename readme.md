# ts_remap_coverage

- Test framework
    + mocha
- Coverage 
    + istanbul
    + remap-istanbul

## Getting started

```sh
# install library
$ npm i

# Run test
$ npm t

# Open coverage/html-report/index.html
```

## Tests
- Examples use `power-assert` via `mocha`
- Explore your favortite assertion library
    + chai, sinon, etc
    + make sure to install the appropriate typings
